Formulário de contratação para planos fixos
Preencha o formulário para contratação de diárias RECORRENTES Super Esposa.

Dentro do tempo escolhido você poderá utilizar para limpeza e/ou para passar roupas (NÃO lavamos roupas). Você deverá distribuir a limpeza conforme suas prioridades, ciente de que a diarista realizará tudo o que for possível no tempo contratado.
É direito das diaristas realizar 15 minutos de pausa para os atendimentos de 6h, e 1 hora de almoço para os atendimentos de 8h, sem prejuízo das horas contratadas.

Se desejar mais de uma diarista por atendimento basta selecionar o número desejado abaixo, lembrando que o valor da diária será multiplicado pelo número de diaristas selecionadas.

A cobrança ocorrerá de forma quinzenal de acordo com o tempo e a frequência selecionados, com vencimentos para todo dia 05 e 20 de cada mês. Os atendimentos terão o valor da diária multiplicado pela quantidade de diárias efetuadas na quinzena.O USUÁRIO concorda com os ajustes e/ou alterações dos preços das diárias sempre que a SUPER ESPOSA SÃO CARLOS julgar necessário, conforme estabelecido nos Termos de Uso. Os valores das diárias estarão sempre disponíveis para consulta no site.

A SUPER ESPOSA SÃO CARLOS se responsabiliza em repor ao USUÁRIO outro dia e horário, previamente combinados, se deixar de atender com suas obrigações determinado agendamento, por motivo de força maior. De igual modo, o USUÁRIO, caso necessite desmarcar um determinado agendamento, deverá fazê-lo com 24 HORAS DE ANTECEDÊNCIA, ocasião em que já definirá junto à SUPER ESPOSA SÃO CARLOS qual a próxima data ideal para a reposição dos serviços, sob a pena de receber a cobrança normal.

Os produtos domésticos destinados à execução dos serviços ficarão a cargo do USUÁRIO que se responsabilizará pelo fornecimento dos mesmos. As diaristas NÃO utilizarão produtos químicos de uso profissional, apenas produtos de uso doméstico.

Ex.: Local das chaves, distribuição das prioridades, localização dos produtos, etc.

How to clone into other project?

  • Press "Ctrl + E" or "Cmd + E" in the Designer and enable "Select on-page element".

  • Search for the class named, "Cloneable Area".

  • Copy the element inside this container to your own project.

  • Rename the classes accordingly if they are clashing with your project.

  • Ensure custom code from the in-page setting has been copied into your project as well (if there's any).

  • 'Page Trigger' interaction can't be copied directly. You can only clone the whole project, or rebuild it.
    A simple trick to copy the 'Page Trigger' interaction into another project is as below:

    1. Create a dummy element.
    2. Apply any type of 'Element trigger' into the dummy element and select the 'Page Trigger' animation.
    3. Copy the dummy element with the animations applied into your new project.
    4. The animation should have been copied into your project and you can reapply the 'Page Trigger' animation into your project.
